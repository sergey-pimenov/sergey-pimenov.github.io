ЗАКРИТИ

Я front-end розробник з Мінська з досвідом близько 2-ох років. Пристрасно захоплений вебом, інтерфейсами зокрема. Для мене це не тільки робота, а й хобі, спосіб відмінно проводити час.

Список моїх навичок:

  • HTML5 Семантичний, валідний. Без брудних хаков, написаний так, щоб був зрозумілий людям і пошуковим роботам.
  • CSS3 Вичавлюю з CSS все. Flexbox, складні анімації - без проблем! Те що можна робити за допомогою CSS, робиться саме за допомогою нього.
  • SVG Для мене це не просто розширення для векторних зображень. Це мова вектора з фантастичними можливостями, які я інтенсивно використовую.
  • JavaScript Нативний, ясний для розуміння тих, хто працює з ним. Все що потрібно - коментується, оптимізується. Все як має бути.
  • Sass Як препроцесор використовую Sass. Так що ви можете розраховувати на стилі, з якими приємно працювати.
  • Jade Для того щоб проект було зручніше підтримувати, застосовую шаблонизатор / препроцесор Jade (pug.js). Вихідні тексти з ним стають чистішими і логічніше.
  • Gulp Працюю з даними збирачем. Роблю конфіги до gulp'у так, щоб вони були зрозумілі і коментую їх для зручної і швидкої настройки.
  • Git Потрібно працювати з Git? Без проблем!
  • Адаптивність Роблю так, щоб все було бездоганно на всіх екранах, на всіх дозволах. Можу відмінно адаптувати настільні версії без макетів, для мене це звичайна практика.
  • Анімація / візуалізація для web'а Це мій особливий навик. Анімація прийшла і міцно влаштувалася в web'е. Тепер тільки потрібно застосовувати її якісно і зі смаком. Даному пункту я приділяю особливу увагу, це моє хобі. DOM-анімація, canvas-анімація, CSS3-анімація. Все це мої улюблені технології.
  • SMACCS Як архітектуру для стилів я вибираю SMACCS. Ви можете розраховувати на масштабовану і логічну архітектуру стилів.
  • Архітектура проекту Ви можете бути впевнені в тому, що архітектура клієнтської частини буде продуманої і задокументованої.
