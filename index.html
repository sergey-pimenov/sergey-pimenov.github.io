Я front-end разработчик из Минска с опытом около 2-ух лет. Страстно увлечён вебом, интерфейсами в частности. Для меня это не только работа, но и хобби, способ отлично проводить время.

Список моих навыков:

  • HTML5 Семантичный, валидный. Без грязных хаков, написанный так,чтобы был понятен людям и поисковым роботам.
  • CSS3 Выжимаю из CSS всё. Flexbox, сложные анимации - без проблем! То что можно делать с помощью CSS, делается именно с помощью него.
  • SVG Для меня это не просто расширение для векторных изображений. Это язык вектора с фантастическими возможностями, которые я интенсивно использую.
  • JavaScript Нативный, ясный для понимания тех, кто работает с ним. Всё что нужно - комментируется, оптимизируется. Всё как должно быть.
  • Sass В качестве препроцессора использую Sass. Так что вы можете рассчитывать на стили, с которыми приятно работать.
  • Jade Для того, чтобы проект было удобнее поддерживать, применяю шаблонизатор/препроцессор Jade(pug.js). Исходники с ним становятся чище и логичнее.
  • Gulp Работаю с данным сборщиком. Делаю конфиги к gulp'у так, чтобы они были понятны и комментирую их для удобной и быстрой перенастройки.
  • Git Нужно работать с Git? Без проблем!
  • Адаптивность Делаю так, чтобы всё было безупречно на всех экранах, на всех разрешениях. Могу отлично адаптировать настольные версии без макетов, для меня это обычная практика.
  • Анимация / визуализация для web'а Это мой особый навык. Анимация пришла и прочно обосновалась в сети. Теперь только нужно применять её качественно и со вкусом. Данному пункту я уделяю особенное внимание, это моё хобби. DOM-анимация, canvas-анимация, CSS3-анимация. Всё это мои излюбленные технологии.
  • SMACCS В качестве архитектуры для стилей я выбираю SMACCS. Вы можете рассчитывать на масштабируемую и логичную архитектуру стилей.
  • Архитектура проекта Вы можете быть уверены в том, что архитектура клиентской части будет продуманной и задокументированной.
